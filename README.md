# 🧘‍♂️ SoulSupport — AI Companion for Mental Clarity

**SoulSupport** is a SwiftUI-powered iOS application that helps users navigate anxiety, loneliness, and other emotional challenges through a friendly AI-powered chat experience.

> “Sometimes you just need someone to listen. SoulSupport is here for you.”

---

🧠 Disclaimer
This app is not a substitute for professional mental health help.
If you are struggling, please reach out to a licensed therapist or mental health professional.

## ✨ Features

- 🧠 Chat with an AI trained to respond with empathy
- 💬 Topic-based conversation starter (Anxiety, Motivation, Loneliness, etc.)
- ⏳ Typing indicator for smooth interaction
- 🔔 Local notifications for daily check-ins
- 📦 Offline message persistence (planned)
- 🧩 Beautiful custom UI using UIKit & SwiftUI
